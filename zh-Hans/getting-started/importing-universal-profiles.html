Importing Universal Profiles

The Web3.bio Profile APIs enable developers to easily and quickly integrate Next ID Universal Profiles from Ethereum (ENS), Lens Protocol, Farcaster, .bit and Next.ID into their applications. These APIs are already integrated into Web3.biosearch and profile services.

This documentation describes the publicly available endpoints of the Web3.bio Profile API, which is a set of RESTful JSON APIs. It explains how to use them and what they return.
