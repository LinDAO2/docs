# 简介

NextID 是下一代属于用户的去中心化网络。目前在 NextID 生态系统中实现了三个组件：

## [ProofService](proof-service/intro.md)

连接两个身份 / 社交服务的双向、密码学绑定的平台。

## KVService

基于 ProofService 记录的简单结构存储服务，无限存储空间，有鉴权和历史，适合分布式搭建。

## RelationService
